# PSUR-Generator


A Python-based project that simulates the preparation of **Periodic Safety Update Reports (PSURs)** used in Pharmacovigilance and Regulatory Affairs.

## Overview

This project analyzes adverse event data and generates an ICH-style PSUR report by evaluating:

* Adverse event exposure rate
* Severity distribution (Mild, Moderate, Severe)
* Top reported adverse events
* Benefit-Risk assessment
* File-based report generation
* Logging and exception handling

## Features

* Object-Oriented Programming (OOP)
* Custom Exceptions
* Lambda-based Sorting
* Logging
* File Handling
* Exposure Rate Calculation
* Benefit-Risk Evaluation

## Regulatory Affairs Relevance

PSURs are essential post-marketing safety documents used to continuously assess the benefit-risk profile of approved medicines. This project demonstrates how Python can be applied to support pharmacovigilance and regulatory reporting workflows.
